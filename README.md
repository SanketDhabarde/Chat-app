# Chat App

## Features:💻
1. Create groups to chat with others
2. Send messages
3. Send emojis
4. Upload images

## Technology used:👨‍💻
1. ReactJs
2. Firebase auth
3. Firebase Firestore to store chats
4. Firebase storage for images
5. Material-ui for icons
